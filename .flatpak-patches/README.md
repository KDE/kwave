Some of Kwave's dependencies need to get compiled into the flatpak. This directory holds the patches needed to compile those dependencies.

The id3lib patches were downloaded from https://sources.debian.org/data/main/i/id3lib3.8.3/3.8.3-18/debian/patches/
I also included the copyright file from https://sources.debian.org/data/main/i/id3lib3.8.3/3.8.3-18/debian/copyright, which states the patches are licensed GPL-2+.
