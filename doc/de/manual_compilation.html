2.3. Compilierung und Installation

Um libkwave auf ihrem System zu installieren, geben sie die folgenden Befehle in dem libkwave-Quellen-Hauptverzeichnis an: 
 % cd libkwave
 % make -f Makefile.dist
 % ./configure
 % make
 % su root -c "make install"
 % cd ..

Wenn libkwave erfolgreich installiert wurde, können sie mit der Compilierung und Installation von Kwave fortfahren. Geben sie das folgendene im Kwave-Quellen-Hauptverzeichnis ein: 
 % cd kwave
 % make -f Makefile.dist
 % ./configure
 % make
 % su root -c "make install"

Seitdem libkwave und Kwave autoconf und automake benutzen, sollten sie keine Probleme mit der Compilierung haben. Falls sie Probleme haben, berichten sie sie bitte der Kwave Mailing-Liste.
