3.1. Zwischenablage

Kwave muss unglücklicherweise selbst eine Zwischenablage implementieren und benutzen. Dies ist notwendig aufgrund der mangelnden Unterstützung für andere Formate (als Text und Bitmaps) in der KDE Umgebung, was jedoch in zukünftigen Ausgaben gelöst werden könnte.

Dennoch hat dies einige Nachteile. So können sie beispielsweise keine Daten zwischen zwei Kwave-Instanzen mittels der Zwischenablage austauschen. Jede Instanz von Kwave benutzt ihre eigene Zwischenablage und deshalb kann dies nicht funktieren.

Falls sie also von der Zwischenablage Gebrauch machen wollen, sollten sie nur eine Instanz von Kwave starten und mehrere Kwave Fenster benutzen.
