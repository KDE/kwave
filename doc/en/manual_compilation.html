2.3. Manual Compilation and installation

In order to compile and install libkwave on your system, type the following in the base directory of the source distribution: 
 % cd libkwave
 % make -f Makefile.dist
 % ./configure
 % make
 % su root -c "make install"
 % cd ..

If libkwave is successfully installed, you can continue and compile and install Kwave itself. Type the following in the base directory of the source distribution: 
 % cd kwave
 % make -f Makefile.dist
 % ./configure
 % make
 % su root -c "make install"

Since libkwave and Kwave use autoconf and automake you should have no trouble compiling it. Should you run into problems please report them to the Kwave mailing list.
