3.1. Clipboard

Kwave unfortunately has to implement and use it's own clipboard. This is due to the lack of support for formats other than text and bitmaps in the KDE environment and might be solved in some future release.

However, this has some disadvantages. So if you want to exchange data between two Kwave instances through the clipboard, you will find out that this is simply not possible. Each instance of Kwave uses it's own clipboard and so that can't work.

So if you want to make use of the clipboard in Kwave, you should only start one Kwave instance and use multiple Kwave windows. That's the way it works!
