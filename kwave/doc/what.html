What is Kwave

This help documentation was last updated on August. 4, 1998 for kwave 0.29 alpha

Kwave is a simple sound editing program for the KDE-Desktop.

Its features include :

  • simple cut, copy and paste functions
  • a simple filter design tools
  • an small editor for additive synthesis
  • labeling of signals
  • some analysis functions such as Fourier Transform

    • Kwave internally uses 24 bit integer sample data at free selectable rate. It is possible to edit multiple channel files, although the audio output is still mixed down to mono.
